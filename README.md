# whos.in

## About
Whos.In is a single page web application built for a small local medical team to track the status of their suite(s) and assistants through a single portal.  

## Built With
- [Bootstrap](http://www.getbootstrap.com)
- [React](http://www.reactjs.org)
- [Redux](https://react-redux.js.org/)
- [MongoDB](https://www.mongodb.com/)
- [Express](http://www.expressjs.com/)
